# ClobberClobber

## Instructions to run:

1- Install dependencies:
Run `chmod +x setup.sh` and then run `./setup.sh`

2- Run the programs using following format:
`python3 main.py <BOARD> <FIRST_PLAYER> <TIME_LIMIT>`
